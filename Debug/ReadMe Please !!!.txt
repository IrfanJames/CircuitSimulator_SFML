Controls:
---------------------------------------------------------
0. You can Drag the Background

1. Hover Cursor to the Left of the window for ToolBar

2. Hold(one element) or Selection mode(N) (multiple elements)

3. (Hold component) then Ctrl + R -> Rotate

4. (Hold component) then Ctrl + C -> Copy

5. (Hold component) then Ctrl + V -> Paste

6. (Hold component) then Del      -> Delete

7. P	PrintScreen; Save as Image(PNG)

8. S	Save Project

9. O	Open (saved) Project

10. Esc   Escape
